# Leetcode Helper

This is a small utilty just to get hand's on the npm package publishing and working with travis, coverall, mocha, chai

## Installation

  `npm install leetcode-helper`

## Usage

    var helper = require('leetcode-helper');

### Functions supported
* min()
* max()
* stack (object)
  * pop()
  * push()
  * empty()
  * peek()
* queue (object)
    * enque()
    * deque()
    * empty()
    * peek()  
* node()
* print()
* random()

## Tests

  `npm test`
